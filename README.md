## 🤸 Quick Start

1. **Prerequisites**
   - Git
   - Node.js
   - npm

2. **Clone the Repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/nike_landing_page.git
   cd nike_landing_page
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Run the Project**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the project.
